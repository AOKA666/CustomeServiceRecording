改一下代码，用户点击“保存”之后，除了自动更新本地 excel 文件，还会自动发送一个包含数据的请求，具体请求可以查看 submit_bot.py。

这个请求中包含 auth_token 和 cookie，要设置为用户可以自己输入的状态，然后保存配置。


请求中 payload 的字段与 start.bat 表格字段的对应关系如下，写在括号里的是要引用表格的字段，不加括号表示固定：
    mobile: (电话),
    true_name: str = "未留名",
    follow_records: str = (项目;校区;备注), 
    city: str = (城市),
    city_id: int = (),
    state: str = (省),
    state_id: int = (),
    country_id: int = 18784,
    gender: int = 115003,
    intent_project_id: int = (项目),
    market_channel_id: int = (校区),
    behavior_id: int = (行为),
    profession_id: int = 117004,
    rule_type: str = "100521538",
    tags_infos: Optional[list] = None,
    begin_time=None,
    end_time=None,
    contact_type=None,
    specified_date=None,
    line_tel_phone: str = "",
    over_sea_area_code: str = "",

其中 follow_records 的值是三个字段用分号隔开，例如：银行招聘考试;美团-大连校区;银行面试

注意到 payload 中有一些字段是用 id 形式展示的，需要get请求对应网址获取：
--------------------
state_id
https://apigateway.gaodun.com/prm/api/v1/regions/sub/get?parentId=18784&level=1

response 的 result 列表中找到 name=(省)的项，获取 id 值，同时获取 shortName，保存备用
--------------------
city_id
https://apigateway.gaodun.com/prm/api/v1/regions/sub/get?parentId=18785&level=2
其中 parentId 要换成前一步 state_id
response 的 result 列表中找到 name=(城市)的项，获取 id 值
--------------------
intent_project_id
https://apigateway.gaodun.com/solon/api/v1/authority/projects/without-conflict-projects

response 的 result 列表中找到 name=(项目)的项，获取 id 值
---------------------
market_channel_id
https://apigateway.gaodun.com/solon/api/v1/channel/online/get-channels

response 的 result 列表中找到 name=(校区)的项，获取 id 值
---------------------
behavior_id

(来源)如果是"400"， behavior_id=246
(来源)如果不是"400", behavior_id=247
---------------------
tags_infos 是一个列表，比较复杂，需要向以下网址发送 post 请求
https://apigateway.gaodun.com/solon/api/v1/online-consultation/get-online-user-scope

传入参数
{intentProjectIdList: [intent_project_id], typeIdList: [12308]}

示例：
obTagsValueCheckedList=[{id: 4444, objectId: null, tagsValueCode: 4444, tagsValueName: "未知", checkStatus: true}]
