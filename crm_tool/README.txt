# 客服记录台 使用说明

## 快速开始

1. 双击 `start.bat` 启动程序
2. 浏览器会自动打开 http://127.0.0.1:5000
3. 点击右上角「⚙ 配置」填写影刀触发地址

---

## 影刀配置步骤

### 第一步：在影刀中开启 HTTP 触发器

1. 打开影刀，进入你的机器人
2. 点击「触发器」→「HTTP触发」→ 开启
3. 复制生成的 URL（形如 http://127.0.0.1:9333/api/v1/robots/xxx/run）

### 第二步：在程序中配置

点击右上角「⚙ 配置」，粘贴上面的 URL，保存。

### 第三步：在影刀中接收数据

程序会以 JSON 格式传递数据，在影刀中可通过以下变量名接收：

| 字段   | 影刀变量名         |
|--------|--------------------|
| 项目   | params["项目"]     |
| 电话   | params["电话"]     |
| 微信   | params["微信"]     |
| 城市   | params["城市"]     |
| 备注   | params["备注"]     |
| 项目标签 | params["项目标签"] |
| 校区   | params["校区"]     |

---

## 常见问题

**触发失败"无法连接影刀"**
→ 请确认影刀客户端正在运行，且 HTTP 触发器已开启

**"请先在配置中填写影刀触发URL"**
→ 点击右上角⚙，填写并保存影刀 URL
